My Skills

HTML(HyperText Markup Language) uses predefined tags and elements to label different parts of content, such as headings, paragraphs, links, and images, allowing browsers to display and interpret web pages effectively.

CSS (Cascading Style Sheets) is a simply designed language intended to simplify the process of making web pages presentable. It allows you to apply styles to HTML documents, describing how a webpage should look in terms of colors, fonts, spacing, and more.

Bootstrap is a widely-used open-source front-end framework for web development. It provides a collection of HTML, CSS, and JavaScript components and tools that enable developers to build responsive, mobile-first websites with ease.

JavaScript is a scripting language that enables you to create dynamically updating content, control multimedia, animated images, and pretty much everything else.In other words, it's the magic behind interactive and engaging websites!

MySQL is an open-source, Relational Database Management System that stores data in a structured format using rows and columns. It's software that enables users to create, manage, and manipulate databases.

Python is a high-level, interpreted, and general-purpose dynamic programming language that focuses on code readability. It's widely used for web development, data analysis, machine learning, and more. In short, Python is a versatile language that makes coding accessible and efficient!

Django is a Python framework that simplifies web development by handling complex tasks, allowing developers to focus on building web applications. It emphasizes reusability of components and comes with features like a login system, database connection, and CRUD operations. In essence, Django makes creating database-driven websites easier and follows the MVT (Model-View-Template) design pattern

ReactJS is a popular JavaScript library for building user interfaces (UIs) on the web. It follows a declarative, component-based approach and optimizes rendering performance using the Virtual DOM (Document Object Model). In essence, React allows developers to create dynamic and interactive web applications!

Git is a version-control system for tracking changes in computer files and coordinating work on those files among multiple people. It is a Distributed Version Control System, meaning it doesn't rely on a central server to store all versions of a project's files, instead, each user has a full history of the project on their own hard drive.

Visual Studio Code (VS Code) is a lightweight, open-source code editor developed by Microsoft for Windows, Linux, and macOS, featuring debugging support, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded Git.

CertainlySublime Text is a sophisticated text editor for code, markup, and prose, loved for its slick user interface, extraordinary features, and amazing performance.
